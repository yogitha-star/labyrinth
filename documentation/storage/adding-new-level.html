Adding and Switching Levels

This document explains how to add a new level in Labyrinth and how to switch between different levels during the game.

Understanding Levels

A level in Labyrinth defines the layout of tiles, paths, and the goal. Each level is created using existing tile definitions and configurations. Before adding a new level, it is recommended to review an existing level to understand the structure.

Adding a New Level

  • Locate an existing level configuration in the project.
  • Copy an existing level to use it as a template.
  • Modify the layout and tiles according to the new level design.
  • Save the new level configuration.

Switching Between Levels

To switch between levels, update the level reference used when the game is initialized. The game loads the selected level configuration when it starts or when a level change is triggered.

Adding a Goal Tile

  • Locate an existing goal tile definition in the project.
  • Ensure that only one goal tile is placed in a level.
  • Position the goal tile at the desired target location.
  • The level is completed when the player reaches the goal tile.
