Adding and Switching Levels

This document explains how to add a new level in Labyrinth and how to switch between different levels during the game.

Understanding Levels

Levels in Labyrinth are defined inside /js/levels.js. Each level is created using the new Level("LevelName", [...]) constructor, where the second argument is a two-dimensional array representing the layout of tiles.

Each row in the array represents a row of tiles in the level. Each element inside the row refers to a specific tile definition (such as door.goal, forest.top, etc.).

Adding a New Level

  • Open /js/levels.js.
  • Locate an existing level function such as createFirstLevel() or createAdvancedLevel().
  • Copy an existing level function and rename it (for example, createMyNewLevel()).
  • Modify the 2D tile array to design your new level layout.
  • Ensure that a starting tile is defined using PlayerStartsAt(theme.start) or a similar start tile.
  • Save the file after making changes.

Switching Between Levels

To switch between levels, update the level creation reference used when initializing the game. Levels are loaded through functions defined in /js/levels.js.

For example, if the game currently uses: 

createFirstLevel()

You can switch to another level by replacing it with: 

createAdvancedLevel()

This change is typically done in the game initialization logic (for example, inside /js/gui.js). The selected level function determines which tile configuration is loaded when the game starts.

Adding a Goal Tile

  • Goal tiles are defined inside /js/levels.js as part of the levelâ€™s 2D array.
  • Depending on the theme, a goal tile may appear as door.goal, food.end, dark.end, or similar.
  • Place the goal tile at the desired position inside the level array.
  • The level is considered complete when the player reaches the goal tile.

The engine does not strictly enforce having exactly one goal tile. If multiple goal tiles are added, all of them will be placed in the level. If no goal tile is defined, the level may not complete as expected. Therefore, it is recommended to include exactly one goal tile per level.
